# BFN Direct Debit
BFN Direct Debit is a plugin for [Shopware 6], which allows you to accept payments using the direct debit payment method. It collects the bank account holder, IBAN, and Swift/BIC code and enriches the order and the customer with this data.

## Usage
### Plugin Installation

1. Download the plugin files from the source (e.g., plugin marketplace, official website).
2. Access your Shopware 6 administration panel.
3. Navigate to "Extensions" and select "My extensions".
4. Click on "Upload plugin" and select the extracted plugin ZIP file.
5. Click on "Upload plugin" to initiate the installation process.
6. Once the installation is complete, the plugin will appear in the list of installed plugins.
7. Activate the BFN Direct Debit plugin by clicking on the "Activate" button next to it.
8. Configure the plugin settings according to your requirements, such as specifying the payment method option and customizing the snippets.

Please note that these installation steps may vary slightly depending on the specific version of Shopware 6 or the source from which you obtain the BFN Direct Debit plugin.

Make sure to review the plugin documentation or contact the plugin developer for any specific instructions or additional steps related to the installation process.

### Plugin Configuration

To configure the BFN Direct Debit plugin in Shopware 6, follow these steps:

1. Access your Shopware 6 administration panel.
2. Navigate to "Extensions" and select "My extensions".
3. Find the plugin "BFN - Direct Debit".
4. Click on three dots and then select the configuration option.
5. Select the payment method you want to enable the direct debit fields for.
6. Save the changes to apply the configuration.
